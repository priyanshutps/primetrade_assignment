# Trader Performance vs Market Sentiment

This project analyzes the relationship between market sentiment (Fear/Greed Index) and trader behavior on Hyperliquid.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

## Approach
- Data cleaning and date alignment
- Feature engineering (PnL, win rate, leverage, trade frequency)
- Sentiment-based behavioral analysis
- Actionable strategy recommendations

## Outcome
The analysis highlights how trader performance and risk-taking behavior vary significantly across sentiment regimes.
